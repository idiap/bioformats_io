1) Go to the project directory where the setup.py file is located and install the project package:
$ python -m pip install -e .

2) OME to NPY
Minimal example:
$ python bioformats_io/bioformats_io.py -i ../../dataset/test_data.npy

4) NPY to OME
Minimal example:
$ python bioformats_io/bioformats_io.py -i ../../dataset/test_data.npy --save_ome True
